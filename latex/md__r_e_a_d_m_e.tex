The base project code used for the Advanced Diploma of Professional Game Development to aid in graphical real-\/time applications.

Bootstrap is a static library that can be linked against to implement real-\/time graphical applications with Open\+GL as the base A\+PI.

To build and run projects ensure that Bootstrap is the first project built, and ensure you have set the Working Directory to the bin folder, i.\+e. {\bfseries{\char`\"{}\$(\+Solution\+Dir)bin\textbackslash{}\char`\"{}}}.

Provided is V\+S2015 project templates that can create a basic 2D project, and another for a 3D project. These are within the Tools sub-\/folder as .zip files. Add these to your Visual Studio Project Template directory, i.\+e {\bfseries{\char`\"{}\+C\+:/\+Users/$<$i$>$username$<$/i$>$/\+Documents/\+Visual Studio 2015/\+Templates/\+Project\+Templates/\+Visual C++ Project/\char`\"{}}}.
